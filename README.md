Automation Scripts

A collection of lightweight automation and systems integration projects.
Each script was built to solve a real problem and designed with reliability, reproducibility, and documentation in mind.

Projects
📂 Media Pipeline – Photo/Video Organizer

Problem: thousands of mixed files, duplicates, and corrupt images.

Solution: Python script that sorts by metadata timestamps, deduplicates with SHA-256 hashing, quarantines errors, and generates audit logs.

Outcome: reduced multi-day manual review to <1 day of automated runtime.

🔋 Battery Safe-Shutdown (coming soon)

Problem: risk of data loss when running servers during blackouts.

Solution: Bash script that monitors system battery % and triggers safe shutdowns. Includes dry-run mode for testing.

Outcome: prevented data corruption and extended hardware longevity.
